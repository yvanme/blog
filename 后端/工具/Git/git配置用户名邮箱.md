# git配置用户名邮箱

## 介绍

[TOC]

**环境：CentOS7**

1、查看git配置信息

```shell
git config --list
```

2、查看git用户名

```shell
git config user.name
```

3、查看邮箱配置

```shell
git config user.email
```

4、全局配置用户名

```shell
git config --global user.name "nameVal"
```

5、全局配置邮箱

```shell
git config --global user.email "eamil@qq.com"
```
